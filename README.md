## Description

NestJS API for tracking meal prep orders using [`NestJS`](https://docs.nestjs.com/), [`Objection.js`](https://vincit.github.io/objection.js/), and [`Knex.js`](https://knexjs.org/)

## Installing dependencies

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
